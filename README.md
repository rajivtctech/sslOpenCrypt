# sslOpenCrypt
sslOpenCrypt is a cross-platform, open-source graphical application that brings the full power of OpenSSL to every user — from a student learning cryptography for the first time, to a company director signing MCA21 filings, to a security engineer managing a private certificate authority to an embedded engineer wanting encryption key-pairs for secure OTA updates. It is built for the world but with a special section describing India's cryptography ecosystem.
Its defining philosophy is transparency: every operation is shown as the equivalent openssl command in a live Command Console. Using sslOpenCrypt is simultaneously using OpenSSL — and learning it.
